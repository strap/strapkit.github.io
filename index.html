STRAP

Write your wearable app in JavaScript.
Deploy natively to Pebble and Android Wear.

Github Docs

Get Started

npm install -g strapkit
app.js
var splashPage = StrapKit.UI.Page();

var card = StrapKit.UI.Card({
title: 'Hello!',
body:'I am alive.'
});

// Adds content to a Page
splashPage.addView(card);

// Tells your wearable app to show
// this page
splashPage.show();
